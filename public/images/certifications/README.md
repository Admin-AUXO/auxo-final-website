# Certification Logos

This directory contains certification and partnership logos displayed in the TrustSection component.

## Required Logos

Place the following SVG logo files in this directory:

1. **aws-partner.svg** - AWS Partner Network logo
   - Recommended size: 120x60px
   - Format: SVG (transparent background)
   - Source: AWS Partner Portal

2. **snowflake-partner.svg** - Snowflake Partner logo
   - Recommended size: 120x60px
   - Format: SVG (transparent background)
   - Source: Snowflake Partner Portal

3. **databricks-partner.svg** - Databricks Partner logo
   - Recommended size: 120x60px
   - Format: SVG (transparent background)
   - Source: Databricks Partner Portal

4. **iso-27001.svg** - ISO 27001 Certification badge
   - Recommended size: 120x60px
   - Format: SVG (transparent background)
   - Source: ISO certification provider

## Usage

Logos are displayed in grayscale by default and transition to full color on hover for a professional appearance.

## Notes

- All logos should be SVG format for scalability
- Ensure logos are optimized and minified
- Verify usage rights and compliance with partner brand guidelines
